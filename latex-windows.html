Latex on windows

Installation

Download and install miktex (~100 meg), and winedt (7 meg). (Winedt is free for a month, but after that you have to pay $40. Consider it an investment in your future.)

Reboot.

Edit

Open your latex file with winedt.

Compile

Click the pdf-latex button () to compile your latex document. An error console, as shown below, will pop up and display progress. If there are no errors it will disappear quite quickly. Press Ctrl + E to open up log in the separate pane. You can then navigate through any warnings or errors by continuing to press Ctrl + E.

Preview

Click the adobe acrobate button directly below the pdflatex button.

Learn more

Try reading the user guide in help.
